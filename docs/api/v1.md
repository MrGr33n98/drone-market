# API v1 Documentation

## Base URL
```
http://localhost:3001/api/v1
```

## Authentication
Currently open for read operations. For write operations, include:
```
X-User-Id: <user_id>
```

---

## Endpoints

### Organizations

| Method | Endpoint | Description |
|--------|----------|-------------|
| GET | `/organizations` | List all organizations |
| GET | `/organizations/:id` | Get organization details |
| POST | `/organizations` | Create organization |
| PUT | `/organizations/:id` | Update organization |
| DELETE | `/organizations/:id` | Delete organization |

**Example Response:**
```json
{
  "id": 1,
  "name": "Energy Corp Brasil",
  "cnpj": "00.000.000/0001-00",
  "status": "active",
  "settings": {},
  "created_at": "2026-03-29T00:00:00Z",
  "updated_at": "2026-03-29T00:00:00Z"
}
```

```json
}
```

```json
{
  "error": "Internal server error"
```

## Testing with cURL

```bash
# List organizations
curl http://localhost:3001/api/v1/organizations

# Get order details
curl http://localhost:3001/api/v1/orders/1

# Create lead
curl -X POST http://localhost:3001/api/v1/saas_leads \
  -H "Content-Type: application/json" \
  -d '{"saas_lead":{"name":"Test","email":"test@test.com","status":"new"}}'

# List missions
curl http://localhost:3001/api/v1/missions
```

## Testing with JavaScript Fetch

```javascript
// Get organizations
fetch('http://localhost:3001/api/v1/organizations')
  .then(res => res.json())
  .then(data => console.log(data));

// Create order
fetch('http://localhost:3001/api/v1/orders', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({
    order: {
      title: 'New Order',
      service_type: 'inspection',
      organization_id: 1,
      budget_cents: 100000
    }
  })
});
```
